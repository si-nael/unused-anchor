# v0.5.2 Generative Causal Universe

Status: completed bounded corrective release on 2026-07-15

## Why This Release Exists

`v0.5.1` proved that exact fields, seas, anchors, World Will, causal action, history, inspection, and replay can execute together. It also made an explanatory metaphor too fundamental: it required every world to begin with named protagonists and an authored story graph. The author clarified that protagonists were introduced to explain the intended richness, not to reduce Bubble to a fiction engine. Removing them would be equally wrong. The required correction is a more general world from which persistent beings, agency, relationships, and narratives can arise when its own laws support them.

`v0.5.2` therefore keeps the `v0.5.1` narrative kernel unchanged as a compatible special case and adds `bubble-anchored-causal-world.v2` plus its executable program/runtime. This is a forward generalization, not a rewrite of the pushed release.

## Ontological Boundary

The actor-neutral kernel requires:

- exact world fields and local state;
- positive- and negative-sea coupling to viability;
- guarded internal laws;
- generic anchors with identity predicates and permitted channels;
- World Will objectives, hard constraints, and anchored interventions;
- causal partial order without a universal clock;
- explicit irreversible history commitments;
- generic emergence criteria with exact witness fields.

It does not require a protagonist, an action vocabulary, or a story field. It also does not forbid them from arising. A protagonist-like entity is a possible persistent, self-maintaining, causally effective structure represented by later laws and criteria. A story is a possible projection of causal history, not the primitive substance of the universe.

## State-Generated Events

Let a world state be an exact product

`x ∈ X = X₁ × ... × Xₙ`.

Each internal law `ℓ` has an exact boolean guard `gℓ(x)` and a transition `Tℓ`. The next autonomous frontier is

`F(x) = { ℓ | gℓ(x) = true and ℓ has not applied in this realization }`.

The runtime does not read a pre-authored event graph. A law entering `F(x)` creates its event. All simultaneously enabled laws apply only when their effects commute under the current exact transition fragment. Simultaneous `set` effects on the same field are conservatively non-commuting; the result is `underdetermined`, never declaration-order selection.

The current bounded kernel uses `once-per-realization` laws. That proves finite state-generated closure without claiming persistent or cyclic dynamics.

## World Will Is Not A Narrator

World Will evaluates exact objective values after autonomous closure. Candidate interventions must:

1. cross an uncut anchor whose identity predicates resolve true;
2. use a permitted intervention kind;
3. avoid every protected internal field;
4. change only condition, boundary, sea, or viability fields directly;
5. close under the world's own internal laws;
6. satisfy post-state hard constraints;
7. improve the exact global objective after explicit intervention cost.

World Will may create a condition under which an internal response becomes lawful. It may not write the protected structure and then call that structure self-generated. Exhaustive finite search preserves equal optima as plural only when plural semantics is authored; otherwise an intrinsic tie remains `underdetermined`.

## Emergence Without Premature Naming

An emergence criterion is an exact predicate `e(x)` plus witness fields. Comparing the initial and realized state yields `absent`, `persistent`, `emerged`, `dissolved`, `undetermined`, or `contradicted`. The core records the causal events that changed witness-bearing fields.

These criteria detect structure; they do not assert that every detected structure is alive, conscious, a protagonist, or a story. Those stronger identities require later persistence, boundary, memory, self-maintenance, counterfactual agency, and relation laws. This prevents both forced fiction and forced erasure.

## Order, Reversibility, And History

Evaluation frontiers and host evaluation order are recorded for reproducibility but are not world time. Semantic order is the causal edge relation between generated events. Reversible laws require exact additive inverse effects. Irreversibility alone does not create a durable history arrow; only an explicit history commit does.

## Canonical Evidence

[`examples/self-organizing-field.world.json`](../../examples/self-organizing-field.world.json) contains no protagonist and no authored causal event list. Exact internal laws produce condensation and self-organization from state. The resulting coherent pattern is detected as emerged. World Will can then alter a boundary condition through an anchor; a separate internal law responds and changes the protected coherent field. Disabling World Will still preserves autonomous structure formation.

The run, inspection, record, and replay tools are:

- `apps/hatchery/run-world.ts`
- `apps/hatchery/inspect-world.ts`
- `apps/hatchery/record-world.ts`
- `apps/hatchery/replay-world.ts`

## Compatibility And Non-Closure

`bubble-anchored-narrative-world.v1` remains valid and tested. It is now understood as one explicit agent-bearing specialization that can be useful when the world genuinely contains those structures. It is not the universal minimum for worldhood.

This release does not complete OB-001. It does not yet derive living identity, persistent agents, relationships, open-ended narrative, cross-world anchor transfer, global sea transport, cyclic closure, or recursive self-construction. It establishes the more faithful substrate on which those concepts can be implemented without installing them as decorative names.

## Completion Audit

The 2026-07-15 completion audit found and closed seven concrete semantic failures before release:

- anchor identity is now re-evaluated after every candidate closes under internal law;
- viability traces separate intrinsic and sea-coupled layers, so exact arithmetic remains locally checkable;
- commuting same-frontier effects preserve every direct co-cause;
- intervention combinations are generated lazily under the declared bound rather than materializing the full power set;
- replay verifies stored-run, program, and full-run integrity;
- unknown anchor cuts and field overrides contradict execution instead of disappearing silently;
- stabilization requires an explicit exact target rather than an implicit host-chosen zero.

Each failure has a focused regression in the anchored causal runtime/world test suites. These corrections close the bounded `v0.5.2` release contract, not the larger emergence and whole-universe obligations above.
