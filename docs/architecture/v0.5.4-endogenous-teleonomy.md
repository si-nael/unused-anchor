# v0.5.4 Endogenous Teleonomic Capacity

Status: completed bounded teleonomic-capacity release on 2026-07-17

## Why This Release Exists

`v0.5.3` can distinguish one unlabeled self-maintaining causal component from a passive repeating pattern. Persistence alone still does not show that the structure has an internally maintained norm or more than one lawfully realized way of preserving it. Adding a field named `goal`, an authored menu of actions, or a host-side chooser would merely declare the missing concept.

`v0.5.4` therefore develops a separate evidence layer over the unchanged persistent causal program. It derives a bounded invariant set from the component's exact recurrent configurations, finds response laws only when they occur in the world's own negative-sea disturbance chain, and suppresses one enabled response event while retaining the same program and law definition to test whether that occurrence was actually necessary. It introduces no entity, agent, protagonist, goal declaration, action list, or story graph.

The result is deliberately named **teleonomic capacity**, not life or agency.

## Bounded Invariant Viability Kernel

For a certified persistent path, let `S` be the derived component and let `pi_S(c)` be the exact projection of causal configuration `c` onto the fields in `S`. If a lasso begins at configuration index `i` and has period `p`, the runtime records

`K_S = { pi_S(c_i), ..., pi_S(c_(i+p-1)) }`.

Every transition in the realized cycle must begin and end in `K_S`. This is a finite, executable invariant subset certified by the current lasso. It is not claimed to be the greatest fixed point, a maximal viability kernel, or a proof over unbounded physical history.

The norm is endogenous in the bounded operational sense that no goal object is authored: the recurrent component configurations themselves supply the exact set whose preservation is tested. The report fixes `authoredGoalDeclaration: false`.

## Affordances As Necessary Causal Responses

A response episode must contain the directed event chain

`negative-sea increase -> structural deviation -> boundary-and-memory-dependent internal response`.

The input and output component configurations must both belong to `K_S`. A response is not certified because its identifier looks action-like. For every factual response event generated by law `a`, the runtime reconstructs the factual closure input, retains the exact program and law `a`, records a same-law nonrealization witness, suppresses only that enabled event and its effects, disables World Will, cuts every anchor, and exhausts every selected lawful continuation. The response occurrence is necessary only when

`for every c' in R_(event a does not occur)(c): pi_S(c') is not in K_S`.

The factual and counterfactual program digests must match, the law definition must remain present, and every selected continuation must carry the event-nonrealization witness. An unchanged continuation makes the response `not-necessary`; a mixture or unresolved causal run remains unresolved. The runtime never turns law deletion or one successful counterfactual branch into evidence about one event in the same world.

Syntactic memory binding is also insufficient. For every factual response occurrence, the runtime performs a separate same-program counterfactual that resets all memory-state fields bound by that response to their pre-formation values, disables World Will, and cuts every anchor. Every selected continuation must omit the factual response event. A tautology such as `memory = memory` therefore cannot masquerade as retained-memory dependence.

## Organic Plurality Without A Selector

Plural capacity requires at least two distinct response laws actually realized in one recurrent causal path. It also requires an internal component field, outside the boundary, memory, identity, and structural roles, whose exact pre-response value sets are nonempty and pairwise disjoint across those laws. A differing label or alternating scalar is still insufficient: each response value must be produced inside its sea-disturbance episode, and each next-cycle discriminator input must be written by an event causally downstream of the preceding response. This proves bounded organic recurrence rather than a detached scheduler that merely advances a phase.

There is no parallel candidate array and no step that selects an affordance. The world's guards make exactly the causally enabled response occur. Reports therefore state both `hostSelection: false` and `hostAffordanceSelector: false`.

This does not yet prove counterfactual deliberation, preference ordering, learning, or an agent choosing between simultaneously available actions. It proves a smaller prerequisite: one persistent structure has more than one internally differentiated, causally necessary way of remaining within its own certified invariant set.

## Autonomy Counterfactual

The entire persistent world is re-executed with World Will disabled and every declared anchor cut. Certification records every lawful path and requires the same component to remain persistent with the identical component-configuration kernel on all of them. One matching path is not enough. This prevents external optimization, anchor intervention, or host path selection from being renamed as the structure's own goal-directed capacity.

World Will may still exist elsewhere in Bubble semantics. It simply cannot be the hidden cause of this bounded certificate.

The current causal kernel does not yet generate endogenous plural selected continuations. Plural selected paths arise from equal World-Will interventions; with World Will disabled, commuting internal laws produce one autonomous closure and non-commuting laws remain `underdetermined`. `v0.5.4` preserves universal path-indexed evidence for future branch semantics but does not invent a branch menu to manufacture a plural autonomy fixture.

Forward note dated 2026-07-17: `v0.5.5` later adds explicit maximal-commuting endogenous branch execution. The paragraph above remains the accurate closed boundary of `v0.5.4`; the later release reuses its universal path contract without rewriting its evidence claim.

## Canonical Period-Two World

[`examples/self-maintaining-field.world.json`](../../examples/self-maintaining-field.world.json) now has one organically connected period-two response cycle. After formation, exact internal phase distinguishes two successive sea-pressure routes:

- route B admits pressure at phase `2`, deviates through phase `3`, and restores the component to phase `0`;
- route A admits pressure at phase `0`, deviates through phase `1`, and restores the component to phase `2`.

The certified kernel therefore contains two component configurations. Both response laws require the same boundary and retained memory but are enabled by different internal phase values. Preventing either enabled response event while retaining its law leaves every selected continuation outside the two-state kernel. Three restoration commits remain in monotone anchor history while the two causal configurations recur, so no universal time or full-history repetition is implied.

## Distributed-Channel World

[`examples/distributed-channel-field.world.json`](../../examples/distributed-channel-field.world.json) supplies an independent positive case with no field, parameter, or trace subject named `phase`. Two coupled internal channel coordinates both change during each sea-caused disturbance and each memory-dependent restoration. Their joint state causally produces the next response condition.

Both channel fields independently satisfy the organic discriminator contract, both responses remain necessary under same-program event nonrealization, and both disappear under same-program memory erasure. The example has its own world and anchor identities and mandatory run/inspect/record/replay artifacts. This does not prove that finite recurrence cannot be indexed externally; it proves that the implementation does not require a privileged scalar phase coordinate.

## Adversarial Boundary

The focused verification rejects or preserves as unresolved:

- passive persistence with only one response;
- a name-only goal claim, because the program has no goal-authoring surface and the norm record fixes that fact;
- a detached phase scheduler, even when the same persistent period and two necessary restoration laws remain;
- a response whose memory binding is syntactic but semantically decorative;
- a period-three cycle, which must satisfy the same contract rather than relying on a period-two fixture shape;
- a distributed two-channel cycle with no scalar phase field, which must satisfy the same contract;
- any response whose ablation leaves a lawful continuation inside the kernel;
- World-Will or anchor puppetry, through the disabled-and-cut autonomy re-execution;
- recurrence search that ends before the period-two lasso is found;
- tampering with the stored kernel or counterfactual evidence.

The persistence boundary test was also generalized so every boundary-crossing law realized anywhere in a multi-state cycle must receive its own semantic ablation witness. A law present only in the second phase can no longer escape the membrane proof.

## Compatibility And Non-Closure

The program remains `bubble-persistent-causal-program.v1`; `v0.5.4` is an analysis and evidence layer, not a schema fork. `v0.5.1` narrative, `v0.5.2` causal, and `v0.5.3` persistence semantics remain executable.

This release does not establish an agent, protagonist, consciousness, desire, deliberative choice, endogenous branching, relationship, story, population, evolution, dynamic schema generation, global sea transport, cross-world anchors, or recursive self-construction. It closes the bounded internally maintained norm plus plural organically regenerated, event-necessary, memory-dependent response prerequisite. OB-001, OB-004, OB-005, and OB-011 remain open or partial.
